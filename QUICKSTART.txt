STEP 1: Install Ollama
------------------------
Download and install Ollama from: https://ollama.com/download

Windows: Download the installer and run it
Linux/Mac: See SETUP.md for terminal commands


STEP 2: Download a Model (Required - First Time Only)
------------------------------------------------------
Open PowerShell/Terminal and run:

    ollama pull llama3.2

This downloads the AI model (about 2GB). Other options:
    ollama pull llama2          (4GB - Good balance)
    ollama pull mistral         (4GB - Fast and smart)
    ollama pull phi             (1.6GB - Lightweight)


STEP 3: Start Ollama with CORS Enabled
---------------------------------------
OPTION A - Windows (Double-click):
    Just double-click: start-ollama-widget.bat

OPTION B - Manual (Any OS):
    Windows PowerShell:
        $env:OLLAMA_ORIGINS="*"; ollama serve
    
    Linux/Mac Terminal:
        OLLAMA_ORIGINS=* ollama serve


STEP 4: Open the Widget
------------------------
